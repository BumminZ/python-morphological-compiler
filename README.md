# README.md
## Python Morphological Lexical Analyzer

### Installation
```python
pip install statistics
```

### Usage
```python
from morphological_lexer import MorphologicalLexer
lexer = MorphologicalLexer()
tokens = lexer.tokenize("your_code_here")
```

### Features
- Token recognition
- Pattern analysis
- Performance tracking

### Requirements
- Python 3.8+
